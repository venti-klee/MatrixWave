# MatrixWave 复刻项目

这是一个基于浏览器的交互式复刻项目，重现了论文 **《MatrixWave: Visual Comparison of Event Sequence Data》**（Zhao 等，CHI 2015）中的核心可视化方法。

## 已实现功能

- 使用波形结构串联相邻步骤的转移矩阵。
- 使用矩阵边缘条形表示各步骤的节点访问量。
- 使用矩阵单元格表示页面之间的转移关系。
- 使用图形尺寸编码访问或转移流量。
- 使用紫色—白色—橙色渐变表示两组数据之间的变化。
- 支持切换数据集、调整流量阈值、节点排序和页面搜索。
- 支持对比模式、单独查看数据集、悬停提示和转移详情。
- 支持在 MatrixWave 与 Sankey 图之间切换，并使用完全一致的数据和视觉编码进行对比。
- 支持悬停联动高亮、点击转移后的蓝色关联路径、步骤范围过滤、滚轮缩放和拖拽平移。
- 使用斜线纹理标记访客离开网站的退场节点。

项目中的点击流数据为确定性生成的合成示例数据，用于展示和测试可视化效果，并非论文使用的 Adobe.com 私有数据。

## 本地运行

项目不依赖第三方软件包。安装 Node.js 后，在项目目录中执行：

```bash
node server.mjs
```

然后在浏览器中打开：

```text
http://127.0.0.1:4173
```

也可以使用：

```bash
npm run dev
```

## 生产构建

执行：

```bash
node build.mjs
```

或者：

```bash
npm run build
```

构建结果会生成在 `dist/` 目录中。

## 可视化编码

- **边缘矩形长度**：页面访问量。
- **矩阵单元格背景色**：两组数据之间的转移流量差异。
- **单元格内部黑方块大小**：两组数据的平均转移流量。
- **紫色**：数据集 B 相比数据集 A 流量下降。
- **橙色**：数据集 B 相比数据集 A 流量上升。
- **接近白色**：两组数据的差异较小。

## MatrixWave 与 Sankey 对比

页面顶部可以切换两种可视化：

- **MatrixWave**：使用矩阵单元格表示页面转移，适合关系密集的数据，能够避免大量连线交叉。
- **Sankey**：使用曲线连接相邻步骤，路径方向更直观，但在转移较多时容易出现视觉遮挡。

两种视图共享数据集、筛选阈值、节点排序、差异颜色、搜索、悬停提示和详情面板，便于进行公平比较。

选择一条转移后，系统会用蓝线显示一条关联主路径。由于演示数据是由聚合转移量生成、没有保存原始用户轨迹，因此该路径按照相邻步骤中的最大流量推断；论文原系统则从原始事件序列中计算精确的路径专属流量。
